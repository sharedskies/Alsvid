# Alsvid
Alsvid is a curated and evolving collection of Python and Julia routines for command line processing and analysis of astronomical and other data.
This repository is an open-source read-only mirror archive. The code is provided under the MIT License for the community's benefit. I am not actively accepting pull requests or monitoring issues here.
For documentation, additional information about the the routines, and recent developments visit [Sharedskies software](https://sharedskies.space/software).

## AstroImageJ
The opensource Java-based astronomical image and data processor [AstroImageJ](https://astroimagej.com) is a versatile cross-platform graphical-interface companion to the supplementary command-line tools of Alsvid. 
## Source files
The code is sorted into primary use categories and the file names describe their function.  Comment text within the file explains more, and execution will prompt with required input if not provided.  For additional support, please see [Sharedskies Software](https://sharedskies.space/software).
